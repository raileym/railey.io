  • While Algebra 1 introduces the concept of the Absolute Value for the first time, Algebra 2, Pre-Calculus, and even Calculus make liberal use of the concept.

  • You want to be nimble with problems involving Absolute Value, not challenged.

  • The Absolute Value treats all expressions the same: the negative parts become positive while the positive parts stay the same.

  • When you work with Absolute Value there are two categories of problems: EQUALITY and INEQUALITY. With EQUALITY, you are looking for specific numbers. With INEQUALITY, you are looking for a range of values.

  • These examples are simple numbers.

    \[\begin{gathered} | 3 | = 3 \ \ \textrm{ positive numbers } \\ \\ |-5 | = 5 \ \ \textrm{ negative numbers } \\ \end{gathered}\]

  • These examples use a linear expression inside the absolute value.

    \[\begin{gathered} | x | = 6 \ \ \textrm{ what values of } x \textrm{ work here? } \\ \\ | 3x-2 | = 8 \ \ \textrm{ what values of } x \textrm{ work here? } \\ \end{gathered}\]

  • These examples involve inequalities, in which case we are looking for a RANGE of values for x.

    \[\begin{gathered} | x | < 6 \ \ \textrm{ what range of values for } x \textrm{ work here? } \\ \\ | 3x-2 | <= 8 \ \ \textrm{ what range of values for } x \textrm{ work here? } \\ \end{gathered}\]

  • A simple number line works great with problems involving inequalities, \(\lt\), \(\le\), \(\gt\), and \(\ge\).

  • Remember that when your problem involves \(\le\) and \(\ge\) the endpoints on your graph are going to be SOLID dots. If your problem involves \(\lt\) and \(\gt\), then the endpoints on your graph are going to be HOLLOW dots.

  • These examples illustrates absolute value expressions on both sides of an inequality.

    \[\begin{gathered} | x | \lt | 2x-6 | \ \ \textrm{ what range of values for } x \textrm{ work here? } \\ \\ | 3x-2 | \le | 2x^2 +1| \ \ \textrm{ what range of values for } x \textrm{ work here? } \\ \end{gathered}\]

  • When absolute value expressions on both sides of an inequality, remember that an absolute value can be re-written as two expressions without the absolute value.

  • I definitely graph these problems so that I understand how the answer looks. These ranges can be tricky.

Define Absolute Value

The definition of the Absolute Value states

\[\begin{equation} |x| = \begin{cases} \ \ \ x, & \text{if } \ x \ge 0 \\ -x, & \text{if } \ x \lt 0 \end{cases} \end{equation}\]

for all real numbers \[x \in \mathbb{R}.\]

{{% func_absolute_valueS %}}

A simple example

For example,

\[\begin{equation} |3| = 3. \end{equation}\]

The number \(3\) is positive, therefore \(|3| = 3\) from the definition of absolute value,

\[\begin{equation} |x| = x \ \ \ \text{when } x \ge 0. \end{equation}\]

On the other hand,

\[\begin{equation} \begin{aligned} |-5| = \ & -(-5) \\ = \ & 5. \end{aligned} \end{equation}\]

The number \(-5\) is negative, therefore \(|-5| = 5\) from the definition of absolute value,

\[\begin{equation} |x| = -x \ \ \ \text{when } x \lt 0. \end{equation}\]

The absolute value of a negative number equals its opposite.

  • You know you are working with an absolute value when you see the two vertical bars \[\begin{equation} \LARGE{ | \ \ \ |. } \end{equation}\]

  • The absolute value of a number makes that number positive.

  • The absolute value of a positive number is the number itself.

  • The absolute value of a negative number is the opposite (negative) of the number.

Property: \(|x| \ = \ c\)

When the absolute value of a variable \(x\) equals a constant \(c\),

\[\begin{equation} |x| = c, \end{equation}\]

then the variable \(x\) equals either of two values, positive \(c\) and negative \(c\),

\[\begin{equation} x = \ \pm \ c. \end{equation}\]

{{< spoiler text="Proof: \(|x| = c \ \ \equiv \ \ x = \pm c\)" >}}

Proof:

From the definition of absolute value, the expression

\[\begin{equation} |x| = c \end{equation}\]

can be re-written as two expressions,

\[\begin{align} |x| = x & = c & \textrm{ when } x \ge 0, \tag{1} \end{align}\]

and,

\[\begin{align} |x| = - x & = - c & \textrm{ when } x \lt 0. \tag{2} \end{align}\]

Combining Eqns. (1) and (2) gives us our two values for \(x,\)

\[\begin{equation} x = \pm c. \end{equation}\]

Consider the expression

\[\begin{equation} |x - 3| = 5. \tag{3} \end{equation}\]

Applying the property

\[\begin{equation} |x| = c \ \ \ \equiv \ \ \ x = \pm c \end{equation}\]

to Eq. (3) expression gives you a new expression,

\[\begin{equation} x - 3 = \pm 5. \tag{4} \end{equation}\]

Re-write Eq. (4) into two separate expressions,

\[\begin{align} x - 3 & = \ + 5, \textrm{ and } \\\\ x - 3 & = \ - 5. \end{align}\]

\[\begin{align} x - 3 & = \ 5 \\ x - 3 + 3 & = \ 5 + 3\\ x & = \ 8, \end{align}\]

and

Consider a second example

\[\begin{equation} | 2x - 3 | = 5. \end{equation}\]

Just as before, you approach this style of problem using the property

\[\begin{equation} |x| = c \ \ \ \equiv \ \ \ x = \pm c. \end{equation}\]

First, rewrite the equation into your new equation

\[\begin{equation} 2x - 3 = \pm 5, \end{equation}\]

which translates into two separate equations

\[\begin{align} 2x - 3 = & \ + 5, \textrm{ and } \\\\ 2x - 3 = & \ - 5. \end{align}\]

Now, solve for \(x\) in each case:

\[\begin{align} 2x - 3 = & \ 5 \\ 2x - 3 + 3 = & \ 5 + 3\\ 2x = & \ 8\\ 1/2 \cdot (2x) = & \ 1/2 \cdot (8)\\ x = & \ 4, \end{align}\]

and

\[\begin{align} 2x - 3 = & \ - 5 \\ 2x - 3 + 3 = & \ - 5 + 3\\ 2x = & \ -2 \\ 1/2 \cdot (2x) = & \ 1/2 \cdot (-2) \\ x= & \ -1. \end{align}\]

Our final solution set is the set

\[\begin{equation} x = \{ -1, 4 \}. \end{equation}\]

Property: \(|x| \lt c\)

When the absolute value of a variable \(x\) is less than a constant \(c\),

\[\begin{equation} |x| \lt c, \end{equation}\]

then you can re-write this expression as a range of values for \(x,\)

\[\begin{equation} -c \lt x \lt c. \end{equation}\]

Here, values for \(x\) fall somewhere between negative \(c\) and positive \(c\).

{{< spoiler text="Proof: \(|x| \lt c \ \ \equiv \ \ -c \lt x \lt c\)" >}}

Proof:

From the definition of absolute value, the expression

\[\begin{equation} |x| \lt c \end{equation}\]

can be written as two expressions,

\[\begin{align} |x| = x& \ \lt c \textrm{ when } x \ge 0, \tag{5} \end{align}\]

and,

\[\begin{align} - |x| = x & \ \lt c \textrm{ when } x \lt 0 \\ |x| = x & \ \gt - c \textrm{ when } x \lt 0. \tag{6} \end{align}\]

Combining Eqns. (5) and (6) gives us our range for \(x\)

\[\begin{equation} -c \gt x \gt c. \end{equation}\]

{{< /spoiler >}}

Consider the following inequality,

\[\begin{equation} | x | \lt 6. \end{equation}\]

By the property

\[\begin{equation} |x| \lt c \ \ \ \equiv \ \ \ -c \lt x \lt c, \end{equation}\]

we expand this inequality to read

\[\begin{equation} -6 \lt x \lt 6. \end{equation}\]

Property: \(|x| \le c\)

When the absolute value of a variable \(x\) is less than or equal to a constant \(c\),

\[\begin{equation} |x| \le c, \end{equation}\]

then you can re-write this expression as a range of values for \(x,\)

\[\begin{equation} -c \le x \le c. \end{equation}\]

Here, values for \(x\) fall somewhere between negative \(c\) and positive \(c\), including the two endpoints negative \(c\) and positive \(c\).

{{< spoiler text="Proof: \(|x| \le c \ \ \equiv \ \ -c \le x \le c\)" >}}

Proof:

From the definition of absolute value, the expression

\[\begin{equation} |x| \le c \end{equation}\]

can be written as two expressions,

\[\begin{align} |x| = x& \ \le c \textrm{ when } x \ge 0, \tag{5} \end{align}\]

and,

\[\begin{align} - |x| = x & \ \le c \textrm{ when } x \le 0 \\ |x| = x & \ \ge - c \textrm{ when } x \le 0. \tag{6} \end{align}\]

Combining Eqns. (5) and (6) gives us our range for \(x\)

\[\begin{equation} -c \ge x \ge c. \end{equation}\]

{{< /spoiler >}}

Consider the following inequality,

\[\begin{equation} | x | \le 6. \end{equation}\]

By the property

\[\begin{equation} |x| \le c \ \ \ \equiv \ \ \ -c \le x \le c, \end{equation}\]

we expand this inequality to read

\[\begin{equation} -6 \le x \le 6. \end{equation}\]
