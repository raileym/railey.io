Pascal’s Identity, also known as Pascal’s Rule, expresses a binomial coefficient in \(n\) as the sum of two binomial coefficients in \(n-1\). \(\definecolor{brightyellow}{RGB}{222,255,0}\) \(\definecolor{brightred}{RGB}{255,0,188}\) \(\definecolor{brightgreen}{RGB}{3,255,0}\) \(\definecolor{brightblue}{RGB}{0,205,255}\) \(\definecolor{brightpurple} {RGB}{133,0,255}\) \(\renewcommand{\redbox}[1]{\fcolorbox{black}{brightred}{#1}}\) \(\renewcommand{\greenbox}[1]{\fcolorbox{black}{brightgreen}{#1}}\) \(\renewcommand{\bluebox}[1]{\fcolorbox{black}{brightblue}{#1}}\) \(\renewcommand{\yellowbox}[1]{\fcolorbox{black}{brightyellow}{#1}}\) \(\renewcommand{\purplebox}[1]{\fcolorbox{black}{brightpurple}{#1}}\) \(\renewcommand{\qedsymbol}{\rule{0.5em}{0.5em}}\)

Theorem 1 (Pascal’s Identity) For integers \(n\) and \(k\), where \(k \le n\), Pascal’s Identity is given by the identity

\[\begin{equation} { n \choose k} = {n-1 \choose k-1} + {n-1 \choose k} \end{equation}\]

Proof

\[\begin{aligned} {n \choose k} = & \ \ { n \choose k} \\[1em] = & \ \ \frac{n!}{k!(n-k)!} \\[1em] = & \ \ \frac{(n-1)!}{(k-1)!(n-1-k)!} \left[ \frac{n}{(n-k)k} \right] \\[1em] = & \ \ \frac{(n-1)!}{(k-1)!(n-1-k)!} \left[ \frac{1}{(n-k)}+ \frac{1}{k}\right] \\[1em] = & \ \ \left[ \frac{(n-1)!}{(k-1)!(n-k)!} + \frac{(n-1)!}{k!(n-1-k)!} \right] \\[1em] = & \ \ {n-1 \choose k-1} + {n-1 \choose k} \\[1em] & & \qedsymbol \end{aligned}\]

When using Pascal’s Triangle, the relationship between these combinations are apparent. For example, suppose \(n=5\) and \(k=2\), then Pascal’s Identity states

\[\begin{aligned} \bluebox{ ${ 5 \choose 2}$ } = \ \ & \greenbox{ ${5-1 \choose 2-1}$ } + \greenbox{ ${5-1 \choose 2}$ } \\[1em] = \ \ & \greenbox{ ${4 \choose 1}$ } + \greenbox{ ${4 \choose 2}$ }. \\[1em] \end{aligned}\]

The pattern is clear in Pascal’s Triangle.

\[\begin{equation}\ \ \ { \fcolorbox{transparent}{transparent}{${0 \choose 0}$} }\ \ \ \end{equation}\]\[\begin{equation}\ \ \ { \fcolorbox{transparent}{transparent}{${1 \choose 0}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${1 \choose 1}$} }\ \ \ \end{equation}\]\[\begin{equation}\ \ \ { \fcolorbox{transparent}{transparent}{${2 \choose 0}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${2 \choose 1}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${2 \choose 2}$} }\ \ \ \end{equation}\]\[\begin{equation}\ \ \ { \fcolorbox{transparent}{transparent}{${3 \choose 0}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${3 \choose 1}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${3 \choose 2}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${3 \choose 3}$} }\ \ \ \end{equation}\]\[\begin{equation}\ \ \ { \fcolorbox{transparent}{transparent}{${4 \choose 0}$} }\ \ \ \ \ \ { \fcolorbox{black}{brightgreen}{${4 \choose 1}$} }\ \ \ \ \ \ { \fcolorbox{black}{brightgreen}{${4 \choose 2}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${4 \choose 3}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${4 \choose 4}$} }\ \ \ \end{equation}\]\[\begin{equation}\ \ \ \ { \fcolorbox{transparent}{transparent}{${5 \choose 0}$} }\ \ \ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${5 \choose 1}$} }\ \ \ \ \ \ { \fcolorbox{black}{brightblue}{${5 \choose 2}$} }\ \ \ \ { \fcolorbox{transparent}{transparent}{${5 \choose 3}$} }\ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${5 \choose 4}$} }\ \ \ \ \ \ \ \ { \fcolorbox{transparent}{transparent}{${5 \choose 5}$} }\ \ \ \ \end{equation}\]
