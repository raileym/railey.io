Absolute Value

The definition of the Absolute Value states,

\[\begin{equation} |x| = \begin{cases} \ \ \ x, & \text{if } \ x \ge 0 \\ -x, & \text{if } \ x \lt 0 \end{cases} \end{equation}\]

{{% func_absolute_valueS %}}

Examples

For example,

\[\begin{equation} |3| = 3. \end{equation}\]

The number \(3\) is positive, and therefore the equality \(|3| = 3\) follows from the definition of absolute value for positive numbers,

\[\begin{equation} |x| = x \ \ \ \text{when } x \ge 0. \end{equation}\]

On the other hand,

\[\begin{aligned} |-5| = \ & --5 \\ = \ & 5. \end{aligned}\]

The number \(-5\) is negative, and therefore the equality \(|-5| = 5\) follows from the definition of absolute value for negative numbers,

\[\begin{equation} |x| = --x \ \ \ \text{when } x \lt 0. \end{equation}\]

Property 1: \(|x| \ = \ c\)

When the absolute value of a variable \(x\) equals a constant \(c\), as in

\[\begin{equation} |x| = c, \end{equation}\]

then

\[\begin{equation} x = \ \pm \ c, \end{equation}\]

which says that the variable \(x\) can equal either positive \(c\) or negative \(c\).

{{< spoiler text="Proof: \(|x| = c \ \ \equiv \ \ x = \pm c\)" >}}

Proof:

From the definition of absolute value, the expression

\[\begin{equation} |x| = c \end{equation}\]

can be re-written as two expressions,

\[\begin{align} |x| = x & = c & \textrm{ when } x \ge 0, \tag{1} \end{align}\]

and,

\[\begin{align} |x| = - x & = - c & \textrm{ when } x \lt 0. \tag{2} \end{align}\]

Combining Eqns. (1) and (2) gives us our two values for \(x,\)

\[\begin{equation} x \pm c. \end{equation}\]

Using Property 1

When you see an absolute value expression involving a variable \(x\) and constant \(c\), you can solve for the variable using Property 1.

For example, consider the expression

\[\begin{equation} |x - 3| = 5. \end{equation}\]

Applying Property 1 to this expression gives you a new expression,

\[\begin{equation} x - 3 = \pm 5. \end{equation}\]

You can re-write this new expression into two separate expressions,

\[\begin{align} x - 3 & = \ + 5, \textrm{ and } \\\\ x - 3 & = \ - 5. \end{align}\]

Solve for \(x\) for each case:

\[\begin{align} x - 3 & = \ 5 \\ x - 3 + 3 & = \ 5 + 3\\ x & = \ 8, \end{align}\]

and

\[\begin{align} x - 3 & = \ - 5 \\ x - 3 + 3 & = \ - 5 + 3 \\ x & = \ - 2. \end{align}\]

Property 2: \(|x| < c\)

When the absolute value of a variable \(x\) is less than a constant \(c\), as in

\[\begin{equation} |x| < c, \end{equation}\]

then you can re-write this expression as a range of values for \(x\)

\[\begin{equation} -c \gt x \gt c. \end{equation}\]

{{< spoiler text="Proof: \(|x| < c \equiv -c \gt x \gt c\)" >}}

Proof:

From the definition of absolute value, the expression

\[\begin{equation} |x| \lt c \end{equation}\]

can be written as two expressions,

\[\begin{align} |x| = x& \ \lt c \textrm{ when } x \ge 0, \tag{3} \end{align}\]

and,

\[\begin{align} - |x| = x & \ \lt c \textrm{ when } x \lt 0 \\ |x| = x & \ \gt - c \textrm{ when } x \lt 0. \tag{4} \end{align}\]

Combining Eqns. (3) and (4) gives us our range for \(x\)

\[\begin{equation} -c \gt x \gt c. \end{equation}\]
