Absolute Value

The definition of the Absolute Value states,

\[\begin{equation} |x| = \begin{cases} \ \ \ x, & \text{if } \ x \ge 0 \\ -x, & \text{if } \ x \lt 0 \end{cases} \end{equation}\]

{{% func_absolute_valueS %}}

Examples

For example,

\[\begin{equation} |3| = 3. \end{equation}\]

The number \(3\) is positive, in which case the equality \(|3| = 3\) follows from the definition of absolute value for positive numbers,

\[\begin{equation} |x| = x \ \ \ \text{when } x \ge 0. \end{equation}\]

On the other hand,

\[\begin{aligned} |-5| = \ & --5 \\ = \ & 5. \end{aligned}\]

The number \(-5\) is negative, in which case the equality \(|-5| = 5\) follows from the definition of absolute for negative numbers,

\[\begin{equation} |x| = --x \ \ \ \text{when } x \lt 0. \end{equation}\]

Property 1: \(|x| \ = \ c\)

When the absolute value of a variable \(x\) equals a number \(c\), as in

\[\begin{equation} |x| = c, \end{equation}\]

then by the definition of the absolute value,

\[\begin{equation} x = \ \pm \ c, \end{equation}\]

which says that the variable \(x\) equals either positive \(c\) or negative \(c\).

Using Property 1

When you see an absolute value expression involving a variable \(x\) and constant \(c\), you can solve for the variable using Property 1.

For example, consider the expression

\[\begin{equation} |x - 3| = 5. \end{equation}\]

Applying Property 1 to this expression gives you a new expression,

\[\begin{equation} x - 3 = \pm 5. \end{equation}\]

You can re-write this new expression into two separate expressions,

\[\begin{align} x - 3 & = \ + 5, and \\\\ x - 3 & = \ - 5. \end{align}\]

Solve for \(x\) for each case:

\[\begin{align} x - 3 & = \ 5 \\ x - 3 + 3 & = \ 5 + 3\\ x & = \ 8, \end{align}\]

and

\[\begin{align} x - 3 & = \ - 5 \\ x - 3 + 3 & = \ - 5 + 3 \\ x & = \ - 2. \end{align}\]
