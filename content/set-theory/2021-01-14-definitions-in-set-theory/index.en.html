1 Set Theory

1.1 Definitions

Definition 1.1 (SET) A SET is a well-defined collection of objects.

We use upper-case letters to denote sets: \(A, B, X, Y\).

We represent a set using the symbol \(\{ \ \}\).

Definition 1.2 (ELEMENTS or MEMBERS) The objects in a set are called ELEMENTS or MEMBERS of the set.

We use lower-case letters to denote elements of a set: \(a, b, x, y\).

Given a set \(A\) and an element \(x\), we write

\[\begin{equation} x \in A \end{equation}\]

if \(x\) is an element of \(A\) and

\[\begin{equation} x \not\in A \end{equation}\]

if it is not.

Definition 1.3 (SET ROSTER METHOD) We may represent the members of a set using the set roster method, which lists all its elements between curly brackets \(\{ \ \}\), such as

\[\begin{equation} \{1, 2, 3, 4, 5, 6, 7, 8, 9\}. \end{equation}\]

Definition 1.4 (SET BUILDER NOTATION) We may represent the members of a set using the set builder notation, which describes some property held by all members of the set, such as

\[\begin{equation} \{x | x \text{ is a positive integer divisible by 3}\}. \end{equation}\]

Definition 1.5 (CARDINALITY) The number of distinct elements in a set \(A\) is called the CARDINALITY of \(A\), denoted as \(|A|\).

The cardinality of a set may be finite or infinite.

Definition 1.6 (SUBSET) Given two sets \(A\) and \(B\), if every element of set \(A\) is also an element of set \(B\), then we say \(A\) is a SUBSET of \(B\), and we write

\[A \subseteq B \ \ \equiv \ \ \{x : x \in A \rightarrow x \in B\}\]

However, if there is at least one element of set \(A\) that is not an element of set \(B\), then we say \(A\) is NOT A SUBSET of \(B\), and we write

\[\begin{equation} A \nsubseteq B \ \ \equiv \ \ \exists x \{x : x \in A, x \notin B\}. \end{equation}\]
Definition 1.7 (SET EQUALITY) Two sets \(A\) and \(B\) are equal, denoted \(A = B\), if and only if \(A \subseteq B\) and \(B \subseteq A.\)

Definition 1.8 (UNIVERSAL SET) All sets under investigation are assumed to be contained
in some larger fixed set called the UNIVERSAL SET, denoted \(\Omega\).

Properties:

Every set \(A\) is a subset of the universal set \(\Omega\),

\[\begin{equation} A \subseteq \Omega \end{equation}\]

Definition 1.9 (SET COMPLEMENT) Given a set \(A\), \(A \subseteq \Omega\), the complement of set \(A,\) denoted \(A^c\), is the set containing all elements in \(\Omega\) but not in \(A\).

\[\begin{equation} A^c = \{x : x \in \Omega \text{ and } x \notin A \} \end{equation}\]

\[\begin{equation} A^c = \Omega - A \end{equation}\]

Definition 1.10 (SET UNION) The UNION of sets \(A\) and \(B,\) denoted \(A \cup B,\) is the set containing all elements in either \(A\) or \(B\) or both.

\[\begin{equation} A \cup B = \{ x : x \in A \text{ or } x \in B \} \end{equation}\]

Definition 1.11 (SET INTERSECTION) The INTERSECTION of sets \(A\) and \(B,\) denoted \(A \cap B,\) is the set containing all elements in both \(A\) and \(B\).

\[\begin{equation} A \cap B = \{ x : x \in A \text{ and } x \in B \} \end{equation}\]
Definition 1.12 (SET DIFFERENCE) The DIFFERENCE of sets \(A\) and \(B,\) denoted \(A \backslash B\) or \(A-B\), is the set of all elements in \(A\) but not in \(B.\)

\[\begin{equation} A \backslash B = \{ x : x \in A \text{ and } x \notin B \} \end{equation}\]

\[\begin{equation} A \backslash B = A \cap \overline{B} \end{equation}\]

\[\begin{equation} A - B = \{ x : x \in A \text{ and } x \notin B \} \end{equation}\]

\[\begin{equation} A - B = A \cap \overline{B} \end{equation}\]

Definition 1.13 (SET SYMMETRIC DIFFERENCE) The SYMMETRIC DIFFERENCE of sets \(A\) and \(B,\) denoted \(A \Delta B,\) is the set of all elements that are in \(A\) or \(B\) but not in both.

\[\begin{equation} A \Delta B = \{ x : x \in A \cup B \text{ and } x \notin A \cap B\} \end{equation}\]

\[\begin{equation} A \Delta B = (A \cap \overline{B}) \cup (\overline{A} \cap B) \end{equation}\] SHOW PROOF - SHOW PROOF - SHOW PROOF

\[\begin{equation} A \Delta B = (A \backslash B) \cup (B \backslash A) \end{equation}\] SHOW PROOF - SHOW PROOF - SHOW PROOF

Definition 1.14 (MUTALLY EXCLUSVE) Two sets \(A\) and \(B\) are called MUTUALLY EXCLUSIVE if they contain no common element,

\[\begin{equation} A \cap B = \emptyset. \end{equation}\]

\[\begin{equation} A \text{ and } B \text{ are mutually exclusive } \equiv A \cap B = \emptyset \end{equation}\]

The terms DISJOINT and MUTUALLY EXCLUSIVE are equivalent.

Definition 1.15 (EMPTY SET) The set containing no elements is called the EMPTY SET, denoted as \(\emptyset.\)

\[\begin{equation} \emptyset = \Omega^c \end{equation}\]

The terms EMPTY SET and NULL are equivalent.
