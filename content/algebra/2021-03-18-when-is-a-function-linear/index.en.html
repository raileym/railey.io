A function is linear if its graph forms a straight-line.

{{< spoiler text="Notes" >}}

See a straight-line graph? Think linear function, as in LINEar function. A linear function is called linear precisely because its graph is a straight line.

The reverse situation is true, too. If the graph of a function is a straight-line, then that function is a linear function. Linear functions and straight-line graphs go hand-in-hand.
