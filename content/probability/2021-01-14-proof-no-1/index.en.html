\(\newcommand{\qedsymbolred} {\color{red} {\rule{0.5em}{0.5em}}}\) \(\newcommand{\qedsymbolblack}{\color{black}{\rule{0.5em}{0.5em}}}\) \(\newcommand{\qedsymbol} {\color{white}{\rule{0.5em}{0.5em}}}\)

Show

\[\begin{equation} Pr(A \cup B) = Pr(A) + Pr(B) - Pr(A \cap B) \tag{1} \end{equation}\]

where the sets \(A\) and \(B\) are subsets of the sample space \(\Omega\), \(A \cup B\) denotes the union of two sets \(A\) and \(B\), \(A \cap B\) denotes the intersection of two sets \(A\) and \(B\), and \(Pr(E)\) is a probability function that assigns a probability \(p\) to an event \(E \subseteq \Omega\) with probability \(0 \le p \le 1\).

For this proof, consider the relationship between the sets \(A \cup B\), \(A\), and \(B\). For example, Figure 1 uses a Venn diagram to illustrate one relationship between these sets.

Figure 1: Typical venn diagram for \(A \cup B\)

The problem with the Venn diagram shown by Figure 1 is that the sets shown are not mutually disjoint. When writing proofs involving probability, you need to work with sets that are mutually disjoint. When the sets are mutually disjoint, then and only then can you use the third condition of a probability function, as given by Eq. (2),

For any finite or countably infinite sequence of pairwise mutually disjoint events \(E_1, E_2, E_3, ...,\)

\[\begin{equation} Pr \left( \bigcup_{i \ge 1}E_i \right) = \sum_{i \ge 1} Pr(E_i). \tag{2} \end{equation}\]

(Mitzenmacher and Upfal 2017, 3)

Figure 2 illustrates a different Venn diagram that uses two mutually disjoint sets to represent the union \(A \cup B\).

Figure 2: The union of \(A \cup B\) using mutually disjoint sets

If we use the two sets as illustrated by Figure 2 then we can apply Eq. (2).

Our strategy is as follows. Given the equality

\[\begin{equation} A \cup B = A \cup ( \overline{A} \cup B) \tag{3} \end{equation}\]

with the two mutually disjoint sets \(A\) and \((\overline{A} \cup B)\), then we can use Eq. (2) to say

\[\begin{equation} Pr(A \cup B) = Pr(A) + Pr(\overline{A} \cap B). \end{equation}\]

Provided we can show

\[\begin{equation} Pr(\overline{A} \cap B) = Pr(B) - Pr(A \cap B), \end{equation}\]

then we can combine our last two expressions to complete our proof.

Prove Eq. (3), repeated below,

\[\begin{equation} A \cup B = A \cup ( B \cap \overline{A} ). \tag{3} \end{equation}\]

We will use the following properties from set theory for our proof,

\[\begin{align} A = & \ A \cap \Omega \ \ \ \textrm{ for any set } A \subseteq \Omega, \tag{Property 1}\\ \Omega = & \ A \cup \overline{A} \ \ \ \textrm{ for any set } A \subseteq \Omega. \tag{Property 2} \end{align}\]

The proof of Eq. (3) follows:

\[\begin{align} A \cup B = & \ (A \cup B) && \textrm{... Identity} \\ = & \ (A \cup B) \cap \Omega && \textrm{... by Property 1} \\ = & \ (A \cup B) \cap (A \cup \overline{A}) && \textrm{... by Property 2} \\ = & \ A \cup (B \cap \overline{A}) && \textrm{... } \cup \textrm{ distributes over } \cap \\ &&\qedsymbolblack \end{align}\]

Show

\[\begin{equation} B = (B \cap A) \cup (B \cap \overline{A}). \tag{4} \end{equation}\]

The proof follows:

\[\begin{align} B = & \ (B) && \textrm{... Identity} \\ = & \ (B) \cap \Omega && \textrm{... by Property 1} \\ = & \ (B) \cap (A \cup \overline{A}) && \textrm{... by Property 2} \\ = & \ (B \cap A) \cup (B \cap \overline{A}) && \textrm{... } \cap \textrm{ distributes over } \cup \\ &&\qedsymbolblack \end{align}\]

Given Eqs. (3) and (4), we can now apply Eq. (2) from the probability function,

\[\begin{equation} Pr(A \cup B) = Pr(A) + Pr(B \cap \overline{A}), \tag{5} \end{equation}\]

and

\[\begin{equation} Pr(B) = Pr(B \cap A) + Pr(B \cap \overline{A}), \tag{6} \end{equation}\]

If we solve for \(Pr(B \cap \overline{A})\) using Eq. (6),

\[\begin{equation} Pr(B \cap \overline{A}) = Pr(B) - Pr(b \cap A), \tag{7} \end{equation}\]

and then substitute this value of \(Pr(B \cap \overline{A})\) from Eq. (7) into Eq. (5), then we obtain our desired result,

\[\begin{align} Pr(A \cup B) = & \ Pr(A) + Pr(B \cap \overline{A}) \\[1em] = & \ Pr(A) + Pr(B) - Pr(B \cap A). \\ \end{align}\]

\[\begin{equation} Pr(A \cup B) = Pr(A) + Pr(B) - Pr(B \cap A) \end{equation}\]
