Number 1

\[\begin{equation} {N \choose n} = \left( \frac Nn \right) {N-1 \choose n-1}. \tag{1} \end{equation}\]

{{< spoiler text="Proof" >}}

When deriving the expected value of a random variable with a hypergeometric distribution, we use the identity Eq. (1) above.

\[\begin{align} {N \choose n} = \ & {N \choose n} \\\\ = \ & \frac{N!} {n! (N-n)!} \\\\ = \ & \frac{N\cdot(N-1)!} {n\cdot(n-1)! (N-n)!} \\\\ = \ & \left( \frac Nn \right) \frac{(N-1)!} {(n-1)! (N-n)!} \\\\ = \ & \left( \frac Nn \right) {N-1 \choose n-1} \ \ \ \ \rule{0.5em}{0.5em} \end{align}\]

For example (Wackerly, Mendenhall, and Scheaffer 2002, 65),

\[\begin{align} \mathbb{E}[X] = \ & \sum_{x \in S} x \frac{{N_1 \choose x}{N_2 \choose {n-x}}}{\color{red} {N \choose n}} \\\\ = \ & \sum_{x \in S} x \frac{N_1 !}{x! (N_1 - x)!} \frac{{N_2 \choose {n-x}}}{\color{red} \left(\frac Nn \right) {N-1 \choose n-1}} \end{align}\]

{{< /spoiler >}}

Number 2

\[\begin{equation} {n \choose m} + {n \choose m+1 } = {n+1 \choose m+1} \tag{2} \end{equation}\]

{{< spoiler text="Proof" >}}

\[\begin{align} {n \choose m} + {n \choose m+1 } = \ & \frac{n!}{m!(n-m)!} + \frac{n!}{(m+1)!(n-m-1)!} \\\\ = \ & \frac{n!}{m!(n-m-1)!} \left[ \frac{1}{(n-m)} + \frac{1}{(m+1)} \right]\\\\ = \ & \frac{n!}{m!(n-m-1)!} \left[ \frac{(m+1) + (n-m)}{(m+1)(n-m)} \right]\\\\ = \ & \frac{n!}{m!(n-m-1)!} \left[ \frac{n+1}{(m+1)(n-m)} \right]\\\\ = \ & \frac{(n+1)!}{m!(n-m-1)!} \left[ \frac{1}{(m+1)(n-m)} \right]\\\\ = \ & \frac{(n+1)!}{(m+1)!(n-m-1)!} \left[ \frac{1}{(n-m)} \right]\\\\ = \ & \frac{(n+1)!}{(m+1)!(n-m)!} \\\\ = \ & \frac{(n+1)!}{(m+1)!((n+1)-(m+1))!} \\\\ = \ & {n+1 \choose m+1 } \ \ \ \ \rule{0.5em}{0.5em} \end{align}\]

{{< /spoiler >}}
