Linear inequality is the same as a linear equation, except

  • Linear inequality uses \(\lt, \gt, \le, \ge\)
  • Linear equation uses \(=\).

Linear Inequality in One Variable

A linear inequality in one variable is an inequality that can be written in the form \[ax+b \lt c\] where \(a\), \(b\), and \(c\) are real numbers and \(a \ne 0\).

Linear Equation in One Variable

A linear equation in one variable is an equatino that can be written in the form \[ax+b = c\] where \(a\), \(b\), and \(c\) are real numbers and \(a \ne 0\).
