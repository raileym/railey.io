How to work with BibTeX file in Hugo-Academic

Last updated on Dec 26, 2020 citations

The citations feature in Hugo-Academic is based on Pandoc’s Markdown feature for citations, although you do not use Pandoc’s commands directly. When working with citations you must

  1. use a bibliographical file (e.g., .bib),
  2. refer to this bibliographical file in your post’s front-matter, and
  3. include one or more in-text citations,
  4. use the nocite feature without in-text citations.

Bibliographic File(s)

To start, you need a bibliographical file. I am using a BibTeX-formatted file to organize the information for my bibliography.

For the sake of convenience, I am using Mendeley Desktop to manage my bibliographic data, including generating updated versions of my bibtex file, as necessary.

I am keeping my BibTeX (.bib) file in my /static directory under a directory named bib,

$(HOME)/static/bib

Mendeley Desktop

Many of my academic friends swear by Mendeley’s product. So I decided to join them, albeit my only goal here is to manage my bibtex files.

Here are a couple of caveats for handling Bibtex files with Mendeley Desktop (on a Mac).

  1. Under Preferences/BibTeX, turn off "Escape LaTeX special characters (…). By not doing so, I found that Mendeley Desktop left pesky escape strings that improperly mangled my URL strings. I found the fix/recommendation on tex.stackexchange.
