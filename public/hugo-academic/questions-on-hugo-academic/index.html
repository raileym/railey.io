Last updated on Dec 20, 2020

These are called Go Templates. So the question is whether I can run Go Templates from within documents for either R Markdown or Markdown syntax.

I want to designate a draft page so that I can edit a page without posting the same page for public consumption?

How do I highlight text in-line?

Clearly, I need to be using highlight.js. I can’t tell whether I should be using highlight.js directly, through Hugo-Academic, or through Hugo. Of course, I couldn’t find any references on this topic (either no one asked or no one knows). Then again, I see examples of in-line text highlighting throughout.

I will start again. Apparently, there is no particular method for changing the color of text in-line. The gross recommendation is to state specific HTML syntax, as in here is some highlighted text. For that matter, here is some other highlighted text, which changes the color of a background, as well.

Check out https://medium.com/@codingdudecom/highlight-text-css-97331a5b71b5
