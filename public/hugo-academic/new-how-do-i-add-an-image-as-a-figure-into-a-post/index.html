This post describes how to insert an image into a post as a figure. Several approaches are shown.

How do I include a figure in my post?

When adding a figure to your post, I recommend that you use the same expression – the {{< figure >}} shortcode by Hugo – to describe both your figure and your link to an image. Figure 1 is an example of one of my figures.

How you invoke the {{< figure >}} shortcode depends on whether your post is a Markdown (.md) file or an R Markdown (.Rmd) file. For Figure 1 above, I used the expression for R Markdown files with the help of the Blogdown command blogdown::shortcode( ),

blogdown::shortcode(
  'figure', 
  src='/negative-binomial-distribution/example-1.png',
  title='<strong>Figure 1.</strong> An example of a set of trials using a random variable with a Negative Binomial Distribution.',
  caption='<strong>This is my caption</strong>')

If you are using a regular Markdown file instead, then use the direct expression for {{< figure >}},

{{< figure
    src="/negative-binomial-distribution/example-1.png"
    title="<strong>Figure 1.</strong> Negative Binomial Distribution."
    caption="This is my caption" >}}

And another example follows:

blogdown::shortcode(
  'figure', 
  src='/images/negative-binomial-distribution/Wackerly-Example-3.14.png',
  title='<strong>Figure 1.</strong> An example of a set of trials using a random variable with a Negative Binomial Distribution.',
  alt='<strong>Figure 1.</strong> An example of a set of trials using a random variable with a Negative Binomial Distribution.',
  class='no-class-listed',
  width='500px')

It appears that caption, attr, and attrlink are not used.

It also appears that the src is reference in $(HOME)/static. In my case above, I qualified the PNG file as a file inside of an images directory,

See src in the call to blogdown::shortcode( ) above. Src is repeated below:

    src='/images/negative-binomial-distribution/Wackerly-Example-3.14.png'

which equates to ...

    $(HOME)/static/images/negative-binomial-distribution/Wackerly-Example-3.14.png

See https://www.rdocumentation.org/packages/blogdown/versions/0.21/topics/shortcode

See https://gohugo.io/content-management/shortcodes/

So where do I put my image?

In my case, I placed my image file – a PNG file – in my website’s static directory further classified under a qualified directory /static/negative-binomial-distribution/,

What about using a Markdown Link \(\mathbf{![\text{title}](\text{url})}\)?

Another approach to adding a figure to your post is to use a Markdown Link for URLs. See Adam Pritchard’s Markdown Cheatsheet as a great reference for using links in a Markdown file.

For example, if I use a Markdown Link

![<strong>Figure 2.</strong> Negative binomial distribution.](/negative-binomial-distribution/example-1.png)

then I get an included figure as shown below in Figure 2,

Figure 2. Negative binomial distribution.

I like using a Markdown Link because the usage is clean and simple. Another great reference on Markdown Links is provided by Gitlab.
